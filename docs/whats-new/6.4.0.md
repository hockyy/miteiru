# What's New in Miteiru 6.4.0

Changes since [v6.3.0](https://github.com/hockyy/miteiru/releases/tag/v6.3.0).

## Features

- **JLPT grammar study on Learn** — When Japanese is active, the left column adds **Random grammar** with an optional JLPT level filter (N5–N1). Picks from a bundled catalog of 831 grammar points (`language-assets/japanese/grammar/jp_grammar.json`).
- **Grammar AI enrichment** — Generate usage notes, example sentences, fun facts, and related patterns via OpenRouter. Notes persist in `user.grammarNotes` per grammar ID.
- **Grammar → analyzer workflow** — Each example uses the same row UI as AI translation (per-sentence **Copy** and **→ Analyzer**). Bulk **Send all** / **Append all to analyzer** pushes examples into the middle column for tokenization and practice.

## Improvements

- **Learn left column** — `LearnStudyPanel` combines AI translation and grammar study (replaces standalone `AITranslationPanel`, which now re-exports for compatibility).
- **Modular grammar modules** — `useGrammarStudy`, `useGrammarNotes`, `useGrammarAiNotes`, `GrammarStudyControls`, `GrammarStudyDisplay`, and `aiGrammarPrompts.ts`.

## Developer notes

- New asset entry in `japanese/asset-manifest.json`: `grammar/jp_grammar.json`.
- Grammar IDs use `{level}-{index}` (e.g. `N5-3`). Edit the JSON catalog directly to add or change entries.
- Tests: `tests/grammarStudy.test.ts`.
