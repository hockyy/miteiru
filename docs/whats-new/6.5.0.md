# What's New in Miteiru 6.5.0

Changes since [v6.4.0](https://github.com/hockyy/miteiru/releases/tag/v6.4.0).

## Features

- **MeaningBox inflection (Japanese)** — Verbs and i/na-adjectives show a conjugation table powered by `kamiya-codec`: essential forms by default, **Show more** for extended auxiliaries, and a **breakdown ladder** when you click an inflected form.
- **Inflection → My Notes** — **Add examples** saves one template sentence per visible form into `user.notes`, with the dictionary gloss plus form label and usage hint.
- **Shared example UI** — Inflection previews and saved note examples use the same `TranslationVariantRow` as grammar/AI translation (sentence, meaning, Copy, **→ Analyzer** on Learn).

## Developer notes

- Main-process wrapper: `main/handler/languages/inflectionTable.ts`; IPC `getInflectionTable`.
- `nextron.config.js` points the `kamiya-codec` webpack external at the CJS build for Electron main.
- Tests: `tests/inflectionTable.test.ts`, `tests/inflectionExamples.test.ts`.
