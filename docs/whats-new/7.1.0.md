# What's New in Miteiru 7.1.0

Changes since [v7.0.0](https://github.com/hockyy/miteiru/releases/tag/v7.0.0).

## Home

- **Main menu revamp** — Watch, Learn, and Flashcards mode cards with a compact language dropdown.
- **Last-used language** — Home remembers and marks your last selected language stack.
- **MeaningBox palette** — Light blue/yellow filled panels matching the dictionary popup style.
- **System & tools panel** — Collapsible Live CC, FFmpeg, and optional media tools; dict cache removal requires **3 confirmations**.

## Learn

- **Live CC + analyzer together** — Live captions and the tokenized analyzer preview show at the same time (no longer mutually exclusive).
- **→ Analyzer** — Copy the current live caption into the middle column with one click.
- **Compact controls** — Smaller live CC toggle, voice picker, and action button grid.

## Keyboard

- **Ctrl+H** — Go home from Learn and Flashcards (aligned with the video player and keyboard help).

## Fixes

- **Build reliability** — Pre-build cleanup of `renderer/.next` and exclude dev cache from packaged builds.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v7.0.0...v7.1.0
