# What's New in Miteiru 7.4.0

Changes since [v7.3.0](https://github.com/hockyy/miteiru/releases/tag/v7.3.0).

## UI

- **Learn study column** — Left column uses shared `MiteiruPanel` / `MiteiruActionBar` components (matches middle/right learn columns).
- **Removed study header** — Dropped the “Study tools / AI translation and JLPT grammar” banner; sections use panel label bars instead.
- **Light purple backdrop** — Study column keeps `from-purple-50 to-pink-50` gradient via new `UI_STUDY_COLUMN_BG` token.
- **Grammar controls** — JLPT level filter and random pick live in a MeaningBox-style panel.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v7.3.0...v7.4.0
