# What's New in Miteiru 7.4.3

Changes since [v7.4.2](https://github.com/hockyy/miteiru/releases/tag/v7.4.2).

## Features

- **Sentence Anki deck** — Learn-page sentence cards export to `Miteiru::{Language}::Sentence` (replacing `::Hard`).
- **AI sentence readings** — Build Anki uses OpenRouter JSON ruby segments (contextual readings like あさって for 明後日); Miteiru builds the ruby HTML locally.
- **HTML export preview** — The Anki export confirmation modal renders front/back previews as HTML, including furigana.

## Fixes

- **Vocabulary Anki export** — When MeaningBox shows multiple headword variants, export uses the first variant only (no concatenated ruby).
- **Grammar Anki export** — One Japanese → English card per grammar point (removed duplicate English → Japanese recall card).

## Developer notes

- `getPrimaryRomajiedVariant`, `getSentenceDeckName`, `buildSentenceRubyHtmlFromSegments`.
- Expanded tests: `ankiExport.test.ts`, `grammarAnkiExport.test.ts`, `parseSentenceAnkiBack.test.ts`.
