# What's New in Miteiru 7.4.2

Changes since [v7.4.1](https://github.com/hockyy/miteiru/releases/tag/v7.4.1).

## Features

- **Vocab sidebar AI notes** — Saved vocabulary (Ctrl+B) shows a **Notes** badge when My Notes exist, a compact preview (definition, usage tip, fun fact, example count), and **AI notes** / **Regenerate AI** shortcuts per word without opening MeaningBox.
- **Notes summary** — Sidebar header and chips show how many saved words have AI notes.

## Developer notes

- Shared `generateUserNoteWithAI` util (MeaningBox + vocab sidebar).
- `hasUserNoteContent`, `getVocabNotePreview`; tests in `tests/vocabNotePreview.test.ts`.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v7.4.1...v7.4.2
