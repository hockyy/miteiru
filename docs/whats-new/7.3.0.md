# What's New in Miteiru 7.3.0

Changes since [v7.2.0](https://github.com/hockyy/miteiru/releases/tag/v7.2.0).

## Fixes

- **Subtitle stroke on Windows** — Outlines now use multi-direction `text-shadow` instead of `-webkit-text-stroke`, which was buggy on Windows/Electron with CJK ruby text.
- **Centralized stroke utility** — `renderer/utils/subtitleStroke.ts` (`getTextShadowFromStroke`, `getSubtitleOutlineStyle`) shared by video subtitles, learn preview, and image popup.
- **Import fix** — Restored missing `getSubtitleTokenPresentation` import in `Subtitle.tsx`.

## Developer notes

- Stroke width slider values still live in `CJKStyling` defaults; they map to shadow radius via `SUBTITLE_STROKE_BASE_WIDTH` (0.22).
- Tests: `tests/subtitleStroke.test.ts`.
- CSS safety net: `.subtitle-stroke-shadow` in `globals.css` disables webkit stroke on subtitle containers.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v7.2.0...v7.3.0
