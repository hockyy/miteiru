# What's New in Miteiru 7.0.0

Changes since [v6.5.0](https://github.com/hockyy/miteiru/releases/tag/v6.5.0).

## Platform upgrades

- **Next.js 16** — Renderer upgraded from Next.js 14 to 16 (`next.config.ts`, Nextron 10).
- **React 19** — React and React DOM bumped to 19.
- **Tailwind CSS 4** — Migrated to Tailwind v4 with `@tailwindcss/postcss` (`postcss.config.mjs`, CSS `@import "tailwindcss"`).
- **Electron 41** — Desktop shell and electron-builder updated for the new toolchain.

## Features

- **Learn vocab sidebar** — Saved vocabulary list on `/learn` (Ctrl+X), matching the video player workflow.
- **Custom buttons** — Replaced `react-awesome-button` with an in-house `Button` component styled to match Miteiru's soft blue UI.

## Breaking changes

- Removed `react-awesome-button` dependency and `AwesomeButton` wrapper.
- Major dependency bumps (Next.js, React, Tailwind, Electron) — rebuild from a clean `node_modules` if upgrading from 6.x.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v6.5.0...v7.0.0
