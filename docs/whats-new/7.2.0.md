# What's New in Miteiru 7.2.0

Changes since [v7.1.0](https://github.com/hockyy/miteiru/releases/tag/v7.1.0).

## UI

- **Shared UI kit** — `renderer/components/UI/` centralizes MeaningBox-style panels, tokens, and conventions (`MiteiruPanel`, `MiteiruActionBar`, `MiteiruSubtitleSlot`, `miteiruUiTheme.ts`).
- **Learn page polish** — Middle and right columns use the same cartoon panel chrome as MeaningBox (yellow label bars, hard offset shadows).
- **Sidebar inputs** — API key and settings fields use light backgrounds with readable text on the dark settings panel.
- **Show/Hide buttons** — Secret-field toggles use the standard secondary `Button` for consistent contrast.

## Fixes

- **API key sync** — OpenRouter key entered in Learning Settings (Ctrl+X) updates Learn/AI tools immediately without leaving the page.
- **Sidebar button layout** — Settings buttons no longer overflow narrow columns (`miteiru-btn--fill`).

## Developer notes

- `meaningBoxTheme.ts` and `homeMenuTheme.ts` re-export from the UI kit for backward-compatible imports.
- Build scripts run `script:clean-next` before all `nextron build` targets.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v7.1.0...v7.2.0
