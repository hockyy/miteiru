# What's New in Miteiru 7.4.1

Changes since [v7.4.0](https://github.com/hockyy/miteiru/releases/tag/v7.4.0).

## Features

- **Export All Anki Cards on Learn** — The bulk My Notes + grammar export is now in the Learn sidebar (Learning Settings → Anki Export), same as on the video settings sidebar.

## Fixes

- **Notes deck no longer duplicates cards** — Terms with My Notes export one card to `Miteiru::{Language}::Notes` (ruby front + note content). The old Easy/Hard pair in the same deck is gone; dictionary-only terms still use separate Easy and Hard decks.

## Developer notes

- Shared hook: `useExportAllAnkiCards` (video + learn sidebars).
- Tests: `tests/ankiExport.test.ts` for Notes vs Easy/Hard card counts.

**Full Changelog**: https://github.com/hockyy/miteiru/compare/v7.4.0...v7.4.1
